让AI代理帮助我们撰写商业计划

## 安装
要安装所需的库,您可以使用pip:

```bash
pip install streamlit
```
```bash
pip install python-docx
```
```bash
pip install openai 
```

## 使用方法
安装库后,您可以使用以下命令运行商业计划生成器应用程序:

```bash
streamlit run streamlit_bussiness.py
```
